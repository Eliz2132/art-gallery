# Art Gallery Website

A beautiful, modern art gallery website built with HTML5, CSS3, and Bootstrap 5. Features responsive design, smooth animations, and an elegant user interface.

## 📋 Project Overview

This project includes a complete art gallery application with:
- **9 Pages Total:**
  1. Login Page
  2. Sign Up Page
  3. Gallery Home Page
  4. Gallery Display Page (with filter functionality)
  5-9. Five Individual Art Piece Pages

## ✨ Features

### Design & Layout
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Bootstrap 5 framework for consistent styling
- ✅ Modern gradient backgrounds
- ✅ Clean and professional UI

### Animations & Interactions
- ✅ Smooth page transitions and fade-in effects
- ✅ Hover animations on cards and buttons
- ✅ Image zoom effects on hover
- ✅ Parallax scrolling effects
- ✅ Button ripple effects
- ✅ Twinkling star background animation
- ✅ Floating background elements
- ✅ Scroll-triggered animations

### Functionality
- ✅ Gallery filtering by category (All, Impressionism, Surrealism, Abstract)
- ✅ Individual art piece detail pages
- ✅ Similar artworks recommendations
- ✅ Smooth navigation between pages
- ✅ Mobile-friendly navigation menu

## 🎨 Pages Description

### 1. **Login Page** (`index.html`)
- Email and password input fields
- Remember me checkbox
- Link to sign up page
- Beautiful gradient background with star animation

### 2. **Sign Up Page** (`signup.html`)
- First name and last name fields
- Email input
- Password and confirm password fields
- Terms and conditions checkbox
- Link to login page

### 3. **Gallery Home Page** (`gallery-home.html`)
- Hero section with call-to-action
- Featured artworks showcase
- About gallery section
- Navigation bar
- Footer

### 4. **Gallery Display Page** (`gallery-display.html`)
- Grid layout showing all artworks
- Category filter buttons (All, Impressionism, Surrealism, Abstract)
- Individual art card with hover effects
- Links to detailed art piece pages

### 5-9. **Individual Art Piece Pages** (`art-piece-1.html` to `art-piece-5.html`)
Each page includes:
- Large artwork image
- Artist name
- Year of creation
- Medium/technique
- Detailed description
- 3 similar artworks with links
- Back to gallery button

## 🚀 How to View the Website

### Option 1: GitHub Pages (Recommended)
Your website is automatically hosted on GitHub Pages:

**URL:** `https://Eliz2132.github.io/art-gallery/`

### Option 2: Clone & Open Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Eliz2132/art-gallery.git
   ```
2. Navigate to the project folder:
   ```bash
   cd art-gallery
   ```
3. Open `index.html` in your web browser:
   - Double-click `index.html` OR
   - Use a local server (recommended):
     ```bash
     python -m http.server 8000
     # or
     python3 -m http.server 8000
     ```
   - Then visit `http://localhost:8000` in your browser

### Option 3: Use VS Code Live Server
1. Install "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"
